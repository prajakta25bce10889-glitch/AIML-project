# TOPIC- AQI(Air Quality Index) PREDICTOR


# PROBLEM STATEMENT-
Air pollution is a serious environmental issue that affects human health.
Poor air quality leads to various health problems such as respiratory diseases, allergies, and long-term lung damage.
The Air Quality Index (AQI) is used to measure the level of pollution in the air. 
However, calculating AQI manually is not easy.
This project aims to make the process of predicting AQI easy using machine learning.

# SOLUTION -
This project provides a simple and interactive way to estimate AQI.
Instead of relying on external datasets or complex formulas, the model is trained on a small set of manually defined sample data
The user is required to enter values for key pollutants such as PM2.5, PM10, NO2, CO, and SO2. 
Based on these inputs, the model predicts an approximate AQI value and categorizes it into a air quality category.

# TECHNOLOGIES USED-
Python – Used as the main programming language for implementation
Scikit-learn – Used for building and training the Decision Tree model

# WHAT THE MODEL REALLY USES-
The model uses a Decision Tree Regressor, which is a machine learning algorithm.
A Decision Tree works by splitting data into smaller groups based on conditions.
It learns patterns from the training data and makes predictions by following those learned rules.
model was chosen because: it is easy to understand and implement and also it works well with small datasets.

# FEATURES-
1.Accepts user input for multiple pollutant values.
2,Predicts AQI instantly based on input.
3.Classifies AQI into categories such as Good, Moderate, and Unhealthy.
4.Does not require any external dataset file.
5.Simple and user-friendly implementation.

# TO RUN THE PROJECT-
1.Install the required library using: pip install scikit-learn.
2.Open the project folder in your terminal or command prompt.
3.Run the program using: python main.py.
4.Enter the pollutant values when prompted.
5.The program will display the predicted AQI and its category.
